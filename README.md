# 🐴 404DonkeyNotFound

> **"If it works, make sure donkey can break IT!"**

An experimental AI-powered web platform that allows:

- 💬 Chat with local LLMs (Ollama)
- 🛠️ Modify the website via web GUI
- 🧠 Use AI to generate and improve code
- 🔄 Backup, rollback, and version control
- 🔧 Full system-level experimentation (unsafe by design)

---

## 🚀 Features

### 💬 Chat Interface
- Connects to Ollama (`http://127.0.0.1:11434`)
- Supports multiple models (e.g. `qwen3:8b`)
- Model selection via UI
- Persistent chat-ready interface

---

### 🧠 AI Development Mode
- Use AI to modify your own website
- Generate frontend/backend code
- Create Zabbix widgets (planned/experimental)
- Load examples to improve responses

---

### 🛠️ Admin Portal
- Manage models
- Edit website files
- Trigger Git sync/push
- Backup and rollback system
- (Optional) Give AI access to system folders
